# ML Project

Machine learning project demonstrating Git and GitHub workflow.

## Files
- `train.py` - Model training script
- `predict.py` - Prediction script
- `utils.py` - Helper functions
- `WORKFLOW_GUIDE.md` - Complete Git workflow documentation

## Setup
```bash
git clone https://github.com/yourusername/ml-project.git
cd ml-project
```

## Usage
```bash
python train.py
python predict.py
```
